# Study AI Bot

Telegram-бот, который:

- просит выбрать класс с 1 по 11;
- спрашивает имя пользователя;
- предлагает выбрать школьный предмет;
- принимает условие текстом или фотографией;
- отправляет задание в Gemini;
- возвращает подробное решение, ответ и простое объяснение;
- хранит пользователя и выбранные данные в SQLite.

## Подготовка

1. Установи Python 3.10 или новее.
2. Открой папку проекта в VS Code.
3. Создай виртуальное окружение:

```bash
python -m venv venv
```

4. Активируй его на Windows:

```bash
venv\Scripts\activate
```

5. Установи библиотеки:

```bash
pip install -r requirements.txt
```

6. Скопируй `.env.example` в новый файл `.env`.
7. Вставь в `.env` токен Telegram-бота и Gemini API-ключ.

## Запуск

```bash
python bot.py
```

## Команды

- `/start` — начать регистрацию заново;
- `/menu` — вернуться в начало.

## Важно

- Не загружай `.env` на GitHub.
- Бесплатные лимиты Gemini всё равно существуют на стороне Google, даже если в самом боте нет пользовательских ограничений.
- Модель можно поменять в `.env` через `GEMINI_MODEL`.
